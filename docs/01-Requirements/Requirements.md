## Overview

The table below are requirements formed based on the needs of Team 307's project. This portion heavily focuses on the Human Machine Interface (HMI). In order to achieve this, I have decided to be focusing on the OLED module which allows users to interact with the machine through the screen. How users would interact with the machinery is by pushing serveral buttons to see the options and if they want to change the speed, they would have three options to choose from: fast, slow, stop. In addition to that, the users can see the data recieved from both the temperature/pressure sensor. The table also focuses on other aspects to make sure communication between teammates is achieved and that this fills the requirements individually.
